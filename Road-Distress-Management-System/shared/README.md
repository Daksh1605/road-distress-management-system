# Shared Module

This folder contains resources shared across frontend, backend, AI, database, and documentation modules.

## Purpose

The shared module helps the team agree on common formats and reusable resources without mixing responsibilities between modules.

## Folder Guide

```text
api-contracts/   API request and response examples
sample-data/     Small sample JSON, CSV, or image metadata files
assets/          Shared diagrams, icons, and common project assets
```

## Suggested Responsibilities

The integration member should focus on:

- Maintaining API contracts
- Providing small sample data for testing
- Coordinating changes between modules
- Helping merge work into `main`
