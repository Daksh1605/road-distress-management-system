# Documentation Module

This folder contains project documentation for development, presentation, and viva.

## Purpose

The documentation module keeps the project easy to understand for guides, evaluators, and team members.

## Folder Guide

```text
proposal/       Problem statement, objectives, scope, and methodology
reports/        Final report, weekly progress, and maintenance report examples
architecture/   System diagrams, data flow diagrams, and module explanations
```

## Suggested Responsibilities

The documentation member should focus on:

- Final year project proposal
- Architecture diagrams
- System workflow explanation
- Viva notes and presentation material
- Keeping documentation aligned with actual implementation
